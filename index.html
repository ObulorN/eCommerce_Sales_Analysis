Overview:

This is a business intelligence data analysis project for Hot Sales eComm, a hypothetical online shop. The goal of the project is to develope a solution that will analyzed the stores perfomance for the year 2003; uncover insights that will aid better decision for increased revenue; using the provided dataset; the Live View for this project is here:


While the dataset for this project can be found here:   dataset

The dataset contains customer related information such as : Order ID, Customer ID, Customer Age, Gender, City, Product Category, Unit Price, Discount_Amount, Total_Amount etc.
